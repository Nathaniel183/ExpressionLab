英语表达功能训练库（本地静态网页）
================================

这是一个面向英语写作与口语训练的离线静态网站。
核心流程：完整范例背诵 → 结构拆解 → 新题迁移 → 补充表达缺口 → 循环复习。

快速启动
--------
Windows：双击 run_windows.bat
macOS / Linux：在终端执行 sh run_mac_linux.sh
随后浏览器会打开或可访问：http://localhost:8000

注意：不要直接双击 index.html。浏览器通常会限制网页读取本地 data 文件，建议始终通过上述本地服务器启动。

主要内容
--------
- 功能覆盖：共享、写作、口语表达功能矩阵
- 核心范例：完整文章、完整回答、功能段落和短片段
- 迁移练习：基础替换、结构迁移、自由表达三级练习
- 补充素材：按功能、语域和标签检索的可插拔语块
- 提示词中心：生成新范例、新练习、新素材、CSS主题和覆盖审计
- 本地进度：掌握、收藏、笔记、自评分；均保存在当前浏览器 localStorage

扩充数据
--------
1. 使用网页“提示词中心”复制所需提示词。
2. 让 AI 严格输出 JSONL 文本。
3. 将结果保存为新的 .txt 文件，放入 data 文件夹。
4. 在 data/manifest.txt 对应数组末尾追加文件名。
5. 刷新网页。

详细说明见 docs/中文使用说明.md 与 docs/数据格式说明.txt。
